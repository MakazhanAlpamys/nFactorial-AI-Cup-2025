# SmartSearch

A smart search engine that combines traditional web search with AI capabilities for enhanced results. The system supports both text queries and image-based search.

## Features

* Text search with AI analysis of results
* Image-based search with intelligent content recognition
* Specialized responses for different query types
* Query optimization through feedback
* "Quick Answer" feature for immediate information
* Advanced adaptive design supporting all device types and dark mode

## Project Structure

* Frontend: React-based user interface
* Backend: Express/TypeScript API service

## Setup Instructions

```bash
# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Run development servers
cd backend
npm run dev

# In a separate terminal
cd frontend
npm start
```
