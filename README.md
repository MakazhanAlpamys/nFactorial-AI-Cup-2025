## Описание проекта

SmartSearch AI - это интеллектуальная поисковая система, которая совмещает традиционный веб-поиск с возможностями искусственного интеллекта для анализа и улучшения результатов. Система поддерживает как текстовые запросы, так и поиск на основе изображений.

## Основные возможности

- Текстовый поиск с AI-анализом результатов
- Поиск на основе изображений с интеллектуальным распознаванием содержимого
- Специализированные ответы для разных типов запросов (определения, инструкции, запросы о личностях)
- Оптимизация запросов с помощью обратной связи
- Предоставление "Быстрого ответа" для оперативного получения информации
- Расширенный адаптивный дизайн с поддержкой всех типов устройств и темного режима

## Архитектура проекта

### Фронтенд (React)
- Компонент поисковой формы с возможностью загрузки изображений
- Компонент отображения результатов с форматированием Markdown
- Отображение анализа AI и "Быстрого ответа"
- Адаптивный дизайн с поддержкой мобильных устройств, планшетов и десктопов
- Оптимизация для сенсорных устройств и поддержка темного режима

### Бэкенд (Express/TypeScript)
- API-интерфейс для текстовых и графических запросов
- Интеграция с поисковыми сервисами (SerpAPI)
- Сервис анализа изображений и оптимизации запросов
- AI-компонент для анализа результатов (на базе Gemini AI)

## Влияние принципов Anthropic на архитектуру

Архитектура SmartSearch AI была разработана с учетом ключевых принципов Anthropic:

### 1. Полезность и безопасность

Система создана, чтобы быть полезной пользователям, одновременно минимизируя потенциальный вред. Это реализовано через:
- Проверку запросов и содержимого изображений перед обработкой
- Прозрачное объяснение происхождения информации
- Предоставление многоуровневого анализа вместо единственного ответа

### 2. Точность и правдивость

- Система использует итеративную оптимизацию запросов для повышения точности результатов
- Анализ включает источники информации и уровень достоверности
- Для неоднозначных запросов предоставляются альтернативные точки зрения

### 3. Сотрудничество между человеком и AI

- Интерфейс разработан для эффективного взаимодействия человека с AI
- Пользователь получает как "Быстрый ответ", так и подробный анализ
- Система учитывает контекст запроса и намерения пользователя

### 4. Специализация и адаптация

- Архитектура включает специализированные анализаторы для разных типов запросов
- Система определяет тип содержимого изображений и применяет соответствующие алгоритмы анализа
- Генерация ответов адаптируется к формату запроса (определения, инструкции, т.д.)
- Адаптивный интерфейс подстраивается под устройство пользователя для оптимального отображения
- Интерфейс учитывает предпочтения пользователя (светлая/темная тема)

### 5. Прозрачность и объяснимость

- Результаты поиска сопровождаются объяснением, как они были получены и проанализированы
- Каждый шаг обработки запроса документирован и доступен для проверки
- Пользователь может видеть, как система интерпретировала его запрос

## Расширенный адаптивный дизайн

Система SmartSearch AI обеспечивает оптимальный пользовательский опыт на устройствах с разными размерами экрана и характеристиками:

### Адаптация к размерам экрана
- **Большие десктопы** (от 1440px): Расширенный интерфейс с многоколоночным отображением результатов и увеличенными элементами
- **Десктопы** (от 992px до 1440px): Полноценный интерфейс с оптимизированной сеткой результатов
- **Планшеты** (от 768px до 992px): Сбалансированный интерфейс с адаптивными компонентами, оптимизированный для среднего размера экрана
- **Малые планшеты** (от 576px до 768px): Интерфейс с одноколоночным отображением результатов и адаптированными элементами управления
- **Мобильные устройства** (от 480px до 576px): Компактный интерфейс с упрощенными элементами и оптимизированной навигацией
- **Малые мобильные устройства** (до 480px): Минималистичный интерфейс с вертикальным расположением элементов, уменьшенным размером шрифтов и оптимизированными элементами управления для сенсорных экранов
- **Сверхмалые экраны** (до 320px): Предельно компактный интерфейс с минимальными отступами и максимальной площадью для контента

### Оптимизация для разных условий использования
- **Темный режим**: Полная поддержка темной темы, активирующейся в соответствии с системными настройками
- **Сенсорные устройства**: Специальные адаптации интерактивных элементов для сенсорного ввода
- **Печать**: Оптимизированные стили для корректной печати результатов поиска
- **Доступность**: Улучшенная поддержка клавиатурной навигации и программ чтения с экрана
- **Производительность**: Оптимизированные анимации и переходы для плавной работы на устройствах с разной мощностью

### Технические особенности
- Использование современного шрифта Inter с оптимизированной загрузкой
- Плавные переходы между состояниями с использованием CSS-переходов
- Гибкая сетка для отображения результатов с использованием CSS Grid
- Семантическая верстка для улучшения SEO и доступности
- Отзывчивый дизайн с использованием относительных единиц измерения

## Установка и запуск

### Требования
- Node.js (версия 16 или выше)
- npm или yarn

### Установка зависимостей

```bash
# Установка зависимостей бэкенда
cd backend
npm install

# Установка зависимостей фронтенда
cd ../frontend
npm install
```

### Запуск проекта в режиме разработки

```bash
# Запуск бэкенда
cd backend
npm run dev

# Запуск фронтенда
cd ../frontend
npm start
```

Фронтенд будет доступен по адресу http://localhost:3000, а бэкенд по адресу http://localhost:3001.

## Технический стек

- **Фронтенд**: React, CSS, адаптивный дизайн, поддержка темной темы, оптимизация для сенсорных устройств
- **Бэкенд**: Node.js, Express, TypeScript
- **AI**: Интеграция с Gemini AI
- **Поиск**: SerpAPI
